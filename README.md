# 2shite-website
AI-driven web app for 2tshie Dental Clinic, Abuja. Developed for CSC 303 (AI Frameworks) by Odumosu Abayomi Alicia. Integrates Gemini for UI/UX, Nano Banana for custom imagery, and Formspree for serverless backend. Featuring a premium Glassmorphism design and deployed via Netlify, it demonstrates full-stack development using modern AI tools.
