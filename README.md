Project Report: AI-Driven Development of 2tshie Dental Web App
1. Executive Summary
This project involved the end-to-end design and deployment of a professional dental clinic application for 2tshie Dental Clinic (Novare Apo Mall, Abuja). The primary objective was to demonstrate the efficacy of Generative AI Frameworks in streamlining the software development lifecycle (SDLC), from data extraction and asset generation to frontend engineering and serverless backend integration.

2. Research & Data Sourcing (Knowledge Acquisition)
The project utilized real-world business data extracted from Google Maps. By feeding raw data regarding clinic location, 24-hour service availability, and specific dental procedures into the Gemini AI Framework, I was able to generate contextually accurate web copy and service descriptions. This showcased the AI’s ability to act as a research analyst and content strategist.

3. UI/UX Design & Image Generation (Prompt Engineering)
The visual identity of the app was crafted using Nano Banana (AI Image Model).

Asset Creation: I engineered specific prompts to generate culturally representative, high-resolution imagery of a Black female dentist and patient, ensuring local relevance.

Design Framework: I implemented a Glassmorphism UI style. This was achieved by directing Gemini to apply complex CSS attributes such as backdrop-filter: blur() and Tailwind CSS transparency layers to create a premium "frosted glass" aesthetic.

4. Technical Implementation (Frontend & JavaScript)
I used HTML5, Tailwind CSS, and JavaScript to build the application.

JavaScript Interactivity: I implemented AI-refined scripts to handle dynamic UI elements, including a scroll-depth-triggered "Back to Top" button and smooth navigation anchors.

Iconography: Integrated Lucide AI vector icons to provide intuitive visual cues for specialized services (e.g., Pediatric Care, Oral Surgery).

5. Backend Infrastructure (Serverless Framework)
To solve the backend requirement without traditional server-side coding, I integrated Formspree as a Form-as-a-Service (FaaS) provider.

Workflow: The application sends POST requests to Formspree, which manages data storage, spam filtering (via AI Akismet), and real-time email notifications to the clinic administrator. This demonstrated a modern Serverless Architecture.

6. Version Control & Deployment
The project followed professional software standards through:

Version Control: Managed via Git and hosted on GitHub, maintaining an audit trail of the development process.

Cloud Hosting: Deployed via Netlify, utilizing their Global CDN and Edge Computing infrastructure for high-speed delivery in Abuja.
